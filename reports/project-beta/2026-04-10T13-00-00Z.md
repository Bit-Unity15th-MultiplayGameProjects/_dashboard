# project-beta — 초기 리뷰

## 1. 주요 변경사항
- 탑다운 2D 레이싱 컨셉 확정, URP 2D Renderer 세팅.
- `Car2DController.cs`: Rigidbody2D 기반 가/감속, 조향, 드리프트 계수 분리.
- 트랙 01 완성 (타일맵 기반 콜라이더, 체크포인트 3개).
- 랩 타임 측정 (`LapTimer.cs`) — 체크포인트 순서 검증 포함.
- 입력 System 사용, 게임패드·키보드 양쪽 맵핑 등록.

## 2. 코드 품질 리뷰
차량 물리가 의외로 깔끔하다. `Car2DController`에서 직진 속도와 조향 속도가
서로 다른 감쇠 계수를 가지고 있어 "실제처럼 느껴지는" 핸들링을 확보.

체크포인트 순서 검증 로직이 `LapTimer` 안에 들어있는데, 트랙이 여러 개가 되면
이 로직은 `Track` 데이터 쪽으로 옮기는 게 자연스럽다. 지금 당장은 문제 없음.

커밋 메시지가 매우 명료하다 (`feat:`, `fix:`, `tune:` prefix 사용) — 이후
히스토리 추적이 편해질 구조. 계속 유지 권장.

## 3. 진행도 평가
프로토타입 단계로는 충실하다. 팀 2인 구성이고 역할 분담(프로그래밍 / 트랙
디자인)이 명확해 보인다.
