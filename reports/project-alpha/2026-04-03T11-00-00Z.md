# project-alpha — 초기 리뷰

## 1. 주요 변경사항
- Unity 2D 템플릿으로 프로젝트 생성.
- 타일맵 기반 그리드(`GridMap.cs`) 구현 — 8x8 셀 단위 이동.
- 플레이어 이동 입력 → 그리드 한 칸씩 이동, tween 없이 즉시 이동.
- 블록 푸시 규칙 초안 (`PushableBlock.cs`): 같은 방향에 장애물 없으면 밀림.

## 2. 코드 품질 리뷰
코드량은 적지만 책임이 깔끔하게 나뉘어 있다. `GridMap`은 순수하게 좌표/충돌
판정만 담당하고 이동은 `PlayerController`가 호출하는 구조 — 이 분리는 유지할 것.

`PushableBlock`과 `PlayerController`의 이동 로직이 약간 중복된다. 두 "그리드
위에서 움직이는 것"이 공통 인터페이스(`IGridMover` 등)를 공유하면 이후
확장(적, 이동 플랫폼)이 쉬워진다.

## 3. 진행도 평가
1주차 목표 달성. 팀 1인 프로젝트라 커밋 수는 적지만 밀도는 낮지 않다.

## 4. 다음 권장사항
- `IGridMover` 또는 `GridEntity` 베이스 클래스 도입 검토.
- 레벨 데이터를 하드코딩 대신 ScriptableObject로 정의할 준비.
- 이동 시 tween (0.1~0.15초) 추가하면 플레이 감각이 크게 개선됨.
